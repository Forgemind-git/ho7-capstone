BETA TOOLS — FEATURE OVERVIEW & PRICING
Source: betatools.io/features + betatools.io/pricing
Last captured: 2024-10-30

=======================================================
KEY FEATURES
=======================================================

CORE WORKFLOW
- Kanban, list, calendar, and timeline views (all plans)
- Subtasks with unlimited nesting depth
- Custom fields: text, number, date, dropdown, people, formula
- Recurring tasks with flexible scheduling (daily/weekly/monthly/custom)

AI FEATURES
- AI writing assistant: rewrite, summarise, translate task descriptions
- Smart scheduling: AI suggests optimal due dates based on team workload
- Meeting notes AI: paste meeting transcript, AI extracts action items automatically

COLLABORATION
- Threaded comments on every item
- Live presence indicators (see who's viewing the same task in real time)
- Video messages: record up to 2-minute loom-style videos directly in tasks
- @mention anyone in your workspace or guest users

REPORTING & ANALYTICS
- Velocity reports: track throughput per person per sprint
- Burndown charts: automatic for any project with dates set
- Custom report builder: drag-and-drop report designer, export to PDF or CSV

SECURITY & ADMIN
- SOC 2 Type II certified
- IP allowlisting (Business plan)
- SCIM provisioning for enterprise identity providers
- Custom data retention policies (Enterprise only)
