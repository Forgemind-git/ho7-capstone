OPEN ISSUES
=======================================================

KI-2041 | MOBILE-BUG | Severity: 2
Title: App freezes on iOS 17.4+ when switching between projects rapidly
Affected versions: iOS app 4.2.0–4.2.1
Workaround: Tap the home button and reopen the app.
ETA for fix: v4.2.2 (target: 2024-11-15)

KI-2038 | WEB-BUG | Severity: 3
Title: CSV export excludes custom field values for records created before 2024-06-01
Affected versions: Web app all versions
Workaround: Use the API export endpoint with ?include_legacy=true parameter.
ETA for fix: 2024-12-01 sprint

KI-2035 | PERFORMANCE | Severity: 3
Title: Dashboard load time exceeds 8 seconds for workspaces with 10,000+ records
Affected versions: Web app all versions
Workaround: Filter the view to fewer than 5,000 records before loading.
ETA for fix: Under investigation — no ETA yet

KI-2029 | INTEGRATION | Severity: 3
Title: Slack notifications not delivered when workspace name contains special characters
Affected versions: Slack integration v2.1
Workaround: Rename the Slack workspace to use only alphanumeric characters and hyphens.
ETA for fix: 2024-11-20
